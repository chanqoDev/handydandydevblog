# Fullstack Blog Platform

A fullstack blog application with a React frontend and backend API for managing content.

## Features
- Create, read, and manage blog posts
- MongoDB database integration
- REST API for content handling
- React + Material UI frontend
- Dynamic rendering of posts

## Tech Stack
React, Material UI, Node.js, Express, MongoDB

## Status
⚠️ Work in Progress — initial version built, currently being refined

## Notes
Focuses on fullstack architecture, API design, and frontend-backend integration.
